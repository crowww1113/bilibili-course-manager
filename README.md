# B站课程学习管理

一个本地使用的 B 站课程学习管理工具，适合用来整理课程列表、记录视频学习进度、统计学习时长，并配合学习日志做复盘。

项目目前以单个 HTML 文件运行，不依赖后端，打开即可使用。

## 项目特点

- 本地运行
- 支持记录每个视频的完成状态和完成日期
- 支持按范围批量标记
- 支持定位到上次学习进度
- 支持新增集数，适合持续更新的课程
- 有学习计时器与学习日志记录功能
- 支持导出 / 导入数据，方便备份迁移

## 项目文件说明

- `B站课程管理1.3.1.html`：主程序，直接打开即可使用
- `视频选集信息一键复制插件.ijs`：辅助脚本，用于在 B 站视频页面复制选集信息
- `使用说明.pdf`：一些常见的问题解答

## 适用场景

- 系统跟踪自己在 B 站上的课程学习进度
- 管理合集课程、系列教程、长期更新课程
- 统计听课时长与实际学习时长

### 如何使用脚本
1. 安装支持用户脚本的浏览器扩展，例如 油猴
2. 浏览器设置——扩展——扩展设置——开发人员模式，打开开关
3. 在油猴导入`视频选集信息一键复制插件.ijs`文件
4. 打开 B 站视频合集页面
5. 点击页面右下角的 **复制选集信息** 按钮
